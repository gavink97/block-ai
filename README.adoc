== Getting Started

=== Installation
https://addons.mozilla.org/en-US/firefox/addon/ai-block-take-back-control/[Firefox: AMO]

=== Build from source

```
gh repo clone gavink97/block-ai .
cd block-ai
npm install
make eb
make build
```

=== Using block-ai

block-ai will close your current tab if the url matches that of an AI website
